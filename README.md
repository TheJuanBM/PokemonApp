##### Prerrequisitos

`node: v16.13.2`

`npm: v9.4.0`

`ruby: v2.7.5`

`pod: v1.11.3`

##### Variables de entorno

POKEMON_URL=""

IMAGE_POKEMON_URL=""

##### Instalar proyecto

`yarn` o `npm i --legacy-peer-deps`

##### Ejecutar proyecto

`npm run start`

Luego en otra terminal:

Android: `npm run android` IOS: `npm run ios`

##### Generar APK

`npm run generate-apk-dev`
